# Installer Component Incubator

Future ownership:

- release selection, acquisition, and verified cache;
- project-owned installation root;
- registry, transaction, recovery, repair, and uninstall;
- version coexistence and transitions;
- Termux exposure and uv discovery integration.

The installer must become an artifact-only consumer. It must not require producer source, build trees, dependency workspaces, launcher source, or repository-relative staging paths.
