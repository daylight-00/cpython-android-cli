# Epoch 2 Component Incubator

This directory declares the future product boundary while implementation remains in its proven Epoch 1 locations.

```text
components/
  standalone/  future standalone producer and release product
  installer/   future artifact consumer and lifecycle manager
```

The README skeleton is not evidence that code has moved or that either component can build independently. See [`../docs/architecture/COMPONENT_OWNERSHIP.md`](../docs/architecture/COMPONENT_OWNERSHIP.md) and [`../docs/roadmap/EPOCH2_ROADMAP.md`](../docs/roadmap/EPOCH2_ROADMAP.md).
