### Probe mutation diagnosis and repair

The first Gate 2 run matched every semantic closure invariant above but failed the candidate mutation control.

Post-failure comparison found exactly:

```text
file entries before   3280
file entries after    3323
delta                    43

new __pycache__ dirs      4
new .pyc files           39
```

The fresh-process probes used `python -I -S -c ...`. Isolated mode ignores `PYTHON*` environment variables, so the shell-level `PYTHONDONTWRITEBYTECODE` and `PYTHONPYCACHEPREFIX` controls did not reach those child interpreters.

The repair makes the child contract explicit:

```text
python -I -B -S -c ...
```

Because deleting generated bytecode does not necessarily restore directory metadata used by the outer fingerprint, rerun from a fresh promoted candidate:
