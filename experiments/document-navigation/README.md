# Document generated navigation — Phase 3

> **Status:** frozen design and verification record
> **Predecessor:** Phase 2 single current-state authority

This experiment introduces complete deterministic navigation derived from the lifecycle registry and current state.

## Scope

- generate one canonical navigation root;
- generate directory entrypoints for current, documentation governance, decisions, epochs, architecture, plans, contracts, evidence, stages, handoffs, references, and experiments;
- preserve the old Stage 3-F handoff entrypoint as a dated historical snapshot;
- update current views to point to generated navigation;
- verify canonical assignment for every tracked Markdown/JSON document;
- preserve Phase 1, Phase 2, and E2-P3 frozen authorities through immediate-predecessor authority identity rather than recursive live-tree replay.

## Exclusions

- no physical document relocation;
- no rewrite of frozen evidence, contracts, stage records, or dated handoffs;
- no product or experiment result change;
- no selection or publication authorization;
- no mixed-document normalization, which remains Phase 4.
