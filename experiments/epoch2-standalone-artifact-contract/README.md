# Epoch 2 Phase 1: Standalone Artifact Contract

Run from the repository root:

```bash
bash experiments/epoch2-standalone-artifact-contract/run-e2p1-standalone-artifact-contract.sh
```

The experiment freezes a versioned release-envelope contract and verifies a deterministic, deliberately unselectable fixture. It does not build or run CPython.
