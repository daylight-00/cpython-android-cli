## TZDATA_SEARCH_PATH_METADATA

```text

```

Observed unique paths:


Associated key:

```text
TZPATH
```

Frozen interpretation:

> These paths are runtime-relevant search-path metadata, not build residue. Their absence requires behavioral probing rather than immediate classification as harmless stale metadata.

CPython `zoneinfo` searches configured `TZPATH` directories first and falls back to the first-party `tzdata` package when system data is unavailable. Stage 3-A therefore treats zoneinfo data sourcing as a separate non-ELF runtime-boundary question.
